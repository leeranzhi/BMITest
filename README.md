# BMITest
> 一款全新的BMI健康值计算器
>
> Android课程设计作业

使用的开源库

```
    implementation 'org.litepal.android:core:1.4.1'
    implementation 'com.github.PhilJay:MPAndroidChart:v3.0.1'
```

1.点击计算BMI值

- 判空，超出范围判断
- 进行显示结果
- 针对各种结果体质提示音
